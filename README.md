# Discrete_Math_project
CS@UCU first-year project for Discrete Mathematics

http://discalc.pythonanywhere.com/

## Tools:
1. Table of truth
2. Worshell algorithm
3. Relations checker
4. Combinatorics
5. Fibonacci numbers
6. Matrices multiplication
7. Bell/Stirling numbers
8. Relation's composition
