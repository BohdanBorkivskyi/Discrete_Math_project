  • Таблиця істинності

    Таблиця істинності — це математична таблиця, що виражає відповідність між усіма наборами величин змінних формули.
    Для складання та заповнення таблиці істинності потрібно:
    1. Обчислити кількість можливих наборів значень логічних змінних. Якщо формула містить n різних логічних змінних, то можливих наборів значень цих змінних буде . Це число визначає кількість рядків у таблиці істинності.
    2. Обчислити кількість логічних операцій у логічному виразі. Ця кількість плюс кількість логічних змінних визначає кількість стовпців таблиці.
    3. Заповнити перші n стовпців усіма можливими наборами значень логічних змінних.
    4. Заповнити кожний наступний стовпець значеннями, отриманими при виконанні чергової логічної операції. В останньому стовпці таблиці будуть отримані усі можливі значення істинності заданого логічного виразу.

  Інакше кажучи, бінарне відношення з A в B – це якась множина впорядкованих пар, у якій перший елемент пари належить множині A, а другий – множині B. 
    Пишуть aRb якщо
    Якщо а та b знаходяться у відношенні (<, >, <=, >=, =), то пишуть aRb, наприклад на множині {1,3,8} існують такі відношення: (1,3),(1,8),(3,1),(3,8),(8,1),(8,3).
    {(2,6),(4,2),(1,1)} - бінарне відношення.
    Бінарні відношення мають свої властивості:
    • рефлексивність: якщо для будь-якого числа х з відношення, існує пара хRx, то відношення рефлексивне, наприклад: {(2,6),(2,2),(6,6)} - рефлексивне, а {(2,6),(6,6)} - ні.
    • антирефлексивність: якщо для будь-якого числа х з відношення, не існує жодної пари хRx, то відношення антирефлексивне, наприклад: {(2,6),(2,2),(6,6)} - не антирефлексивне, {(2,7),(6,6)} - не антирефлексивне, а {(2,6),(6,7)} {(2,6),(6,6)} - антирефлексивне. Відношення може бути одночасно не рефлексивним і не антирефлексивним, але мати обох власнивостей не може.
    • симетричність: відношення симетричне, якщо для будь-якої пари xRy з відношення існує пара yRx, наприклад: {(2,6),(6,2)} - симетричне, {(2,2),(6,6),(3,3)} - симетричне, а {(2,6),(6,6)} - не симетричне
    • асиметричність: відношення асиметричне, якщо для будь-якої пари xRy з відношення не існує пари yRx, наприклад: {(3,6),(6,3)} - не асиметричне, {(2,2),(6,6),(3,3)} - не асиметричне, а {(2,6),(5,6)} - асиметричне.
    • антисиметричність: відношення антисиметричне, якщо для будь-якої пари xRy з відношення не існує пари yRx такої, що х не дорівнює у. Тобто від асиметричного відношення відрізняється лише тим, що тут можуть бути такі пари, як (1,1),(5,5),(7,7) і т.д. Наприклад: {(2,6),(6,3)} - антисиметричне, {(2,2),(6,2),(3,3)} - не антисиметричне, {(6,5),(5,6)} - не антисиметричне.
    • транзитивність: якщо існують пари xRy та yRz, то існує пара xRz. В такому випадку відношення - транзитивне. {(1,3),(3,4),(1,4),(10,2500)} - транзитивне, а {(10,2),(2,5)} - ні, бо не вистарчає пари (10,5).
    • антитранзитивність: якщо існують пари xRy та yRz, то не існує пара xRz. В такому випадку відношення - антитранзитивне. {(1,3),(3,4),(1,4),(10,2500)} - не антитранзитивне, а {(10,2),(2,5)} - так, бо пари (10,5) в даному відношенні немає.

    Число Белла Bn дорівнює кількості розбиттів множини з n елементів на довільну кількість не порожніх підмножин, що не перетинаються.
    Наприклад, B3 = 5, оскільки існує 5 можливих розбиттів множини 
    {a, b, c}
            
            : { {a}, {b}, {c} }1, { {a, b}, {c} }2, { {a, c}, {b} }3, { {a}, {b, c} }4, { {a, b, c} }5
    Додатково вважаємо, що B0 = 1.

  • S(n, n) = 1, n >= 0
